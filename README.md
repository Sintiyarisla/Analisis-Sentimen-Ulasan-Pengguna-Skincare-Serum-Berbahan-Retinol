# Analisis-Sentimen-Ulasan-Pengguna-Skincare-Serum-Berbahan-Retinol
Proyek ini bertujuan membantu calon pengguna lain dalam membuat keputusan yang bijak dengan menyediakan informasi mengenai pengalaman pengguna sebelumnya, termasuk manfaat, risiko, dan aturan pemakaian yang tepat dari skincare serum berbahan retinol.
